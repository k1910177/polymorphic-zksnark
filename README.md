<h1 align="center">Polymorphiz zk-SNARK</h1>

## 設定

ダウンロード
```bash
git clone https://github.com/k1910177/polymorphic-zksnark
```

ビルド
```bash
cargo build --release
```

テスト
```bash
cargo test
```

## 補足
Polymorphic zk-SNARKはArkworksの[ark-groth16](https://github.com/arkworks-rs/groth16)をベースに実装しました。
